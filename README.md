# Alumni Portal Frontend

```sh
```

-   Enter into the directory

```sh
cd alumni-portal-frontend
```

-   Install node modules

```sh
npm i
```

-   Start the local dev server for react

```sh
npm run react
```

-   Visit [localhost:3000](http://localhost:3000) to view the project


### Unauthorized Users

-   Home page
-   Login
-   Register

### Authorized Users

-   Dashboard
-   Profile
-   Settings
-   Memories
-   People you may know
-   Events
